# Code of Conduct

## Our current posture

AIPanel is still an experimental early-stage open-source project.

The expectation here is simple:

- be respectful
- be practical
- assume good intent
- focus on making the project easier to understand, run, and improve

## Expected behavior

Please:

- discuss ideas in good faith
- give clear technical feedback
- keep criticism specific and constructive
- respect maintainers' time and project boundaries
- avoid creating unnecessary drama in issues or pull requests

## Unacceptable behavior

Please do not:

- harass, insult, or demean other people
- post private credentials, secrets, or sensitive personal information
- intentionally derail technical discussion
- demand unpaid support as if it were an entitlement
- use the repo as a place for abusive or hostile behavior

## Scope

This applies to:

- issues
- pull requests
- discussions directly associated with the repo

## Enforcement

Maintainers may:

- ask for discussion to cool down
- close threads that are no longer productive
- reject contributions that violate these expectations
- remove abusive participation if needed

## Practical note

This is not meant to be bureaucratic. The goal is just to keep collaboration useful and low-friction.
